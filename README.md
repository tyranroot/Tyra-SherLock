## 📌 **Overview**

**Tyra SherLock** is a professional-grade Open Source Intelligence (OSINT) tool that allows you to:
- 🔍 **Search usernames** across 50+ social media platforms
- 🖼️ **Analyze images** and perform reverse image searches
- 📊 **Track digital footprints** across the web

> ⚠️ **IMPORTANT**: This tool is for **EDUCATIONAL PURPOSES** only. Use only on publicly available information or with proper authorization.

---

## ✨ **Features**

| Feature | Description |
|:-------:|:------------|
| 🔍 **Username Search** | Search across 50+ social media platforms (GitHub, Twitter, Instagram, Facebook, TikTok, etc.) |
| 🖼️ **Image Analysis** | Extract metadata from images, perform reverse image search via Google/TinEye |
| 🎨 **Hacker Theme** | Matrix background, neon green glow, cyberpunk aesthetic |
| 📱 **Mobile Friendly** | Fully responsive design for all devices |
| ⚡ **Fast & Lightweight** | Pure HTML/CSS/JS, no backend required |
| 🔒 **Privacy First** | Runs entirely in your browser, no data stored |
